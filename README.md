# Smart Alarm Clock

Components:

- [Raspberry Pi 4 Model B (2GB)](https://www.raspberrypi.com/products/raspberry-pi-4-model-b/)
- [7" LCD Display](https://www.waveshare.com/wiki/7inch_HDMI_LCD_(B))

Folder Stucture:
| Folder    | Description        |
|-----------|--------------------|
| app       | React Native App   |
